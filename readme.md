# Scan-Q
Scan-Q is an online attendance management system. The whole project runs on Django.

# Requirement
Any system with python 3 (preferably 3.6) and terminal.

# Usage
- Navigate to the cloned repository.
- Install Django
- Run python manage.py runserver
